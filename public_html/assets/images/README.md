## Image Guidelines

### Quality Standards:
- Use high-resolution images (at least 1920px width for hero images)
- Professional, modern photography
- Avoid overly staged or stock-looking images
- Maintain consistent color grading across all images

### Optimization:
- Compress images before upload (use TinyPNG or similar)
- Target file size: 
  - Hero images: 200-500KB
  - Regular images: 100-300KB
- Use JPEG for photos, PNG for graphics with transparency
- Consider WebP format for better compression

### Free Stock Photo Sources:
1. **Unsplash** (https://unsplash.com)
   - Search: "logistics", "cargo", "shipping", "warehouse"
   
2. **Pexels** (https://pexels.com)
   - Search: "freight", "container", "airplane cargo"
   
3. **Pixabay** (https://pixabay.com)
   - Search: "transportation", "port", "truck"

### Color Theme Compliance:
When selecting images, prefer:
- Navy blue tones (#0B1C2D)
- Orange accents (#F7941D)
- Professional, corporate feel
- Clean, modern aesthetic

## Upload Instructions:
1. Rename downloaded images to match the filenames above
2. Upload to: public_html/assets/images/
3. Verify file permissions are set to 644
4. Test website to ensure images display correctly
